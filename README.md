# Проект "Управление категориями товаров"

## Описание проекта
Проект представляет собой систему для управления категориями товаров и самими товарами. Реализованы базовые классы для создания товаров и категорий с автоматическим подсчетом количества категорий и товаров.

## Реализованные классы

### Класс `Product` (Товар)
Класс для создания объектов товаров с основными характеристиками.

**Атрибуты:**
- `name` (str) — название товара
- `description` (str) — описание товара
- `price` (float) — цена товара
- `quantity` (int) — количество товара на складе

**Методы:**
- `__init__(self, name: str, description: str, price: float, quantity: int)` — конструктор класса, инициализирует товар с переданными параметрами

### Класс `Category` (Категория)
Класс для создания категорий товаров с возможностью группировки товаров.

**Атрибуты класса:**
- `category_count` (int) — счетчик общего количества созданных категорий
- `product_count` (int) — счетчик общего количества товаров во всех категориях

**Атрибуты экземпляра:**
- `name` (str) — название категории
- `description` (str) — описание категории
- `products` (List[Product]) — список товаров в категории

**Методы:**
- `__init__(self, name: str, description: str, products: Optional[List[Product]] = None)` — конструктор класса, создает категорию и автоматически обновляет счетчики категорий и товаров

## Функциональность

### Автоматический подсчет
- При создании новой категории автоматически увеличивается счетчик `category_count`
- При добавлении товаров в категорию автоматически увеличивается счетчик `product_count` на количество товаров
- Счетчики доступны как через сам класс (`Category.category_count`), так и через любой экземпляр категории

### Типизация
Проект использует аннотации типов для лучшей читаемости и поддержки кода:
- `List` — для списков товаров
- `Optional` — для параметров, которые могут быть None
- Явная типизация всех атрибутов и методов

## Пример использования

```python
# Создание товаров
product1 = Product("Samsung Galaxy S23 Ultra", "256GB, Серый цвет, 200MP камера", 180000.0, 5)
product2 = Product("Iphone 15", "512GB, Gray space", 210000.0, 8)

# Создание категории с товарами
category = Category(
    "Смартфоны",
    "Современные смартфоны для повседневного использования",
    [product1, product2]
)

# Получение информации о счетчиках
print(f"Всего категорий: {Category.category_count}")
print(f"Всего товаров: {Category.product_count}")
print(f"Товары в категории: {len(category.products)}")
```

## Реализованная функциональность

### Абстрактный базовый класс BaseProduct
- Создан абстрактный класс для всех продуктов
- Определены общие методы: `__init__`, `__str__`, `__add__`, геттер/сеттер для `price`

### Классы продуктов
- `Product` - основной класс товаров, наследуется от `BaseProduct`
- `Smartphone` - класс для смартфонов, наследуется от `Product`
- `LawnGrass` - класс для газонной травы, наследуется от `Product`

### Миксин ReprMixin
- Добавлено логирование создания объектов
- При создании любого продукта выводится информация о классе и параметрах
- Используется множественное наследование
