# Проект "Управление категориями продуктов"

## Описание
Проект для управления категориями товаров и продуктами с использованием ООП.

## Классы

### Product (Товар)
Базовый класс для всех товаров.
- **Атрибуты**: name, description, __price, quantity
- **Методы**: 
  - `__str__` - строковое представление товара
  - `__add__` - сложение товаров (общая стоимость на складе)
  - `price` (геттер/сеттер) - управление ценой
  - `new_product` - создание товара из словаря

### Smartphone (Смартфон)
Наследник класса Product.
- **Дополнительные атрибуты**: efficiency, model, memory, color

### LawnGrass (Газонная трава)
Наследник класса Product.
- **Дополнительные атрибуты**: country, germination_period, color

### Category (Категория)
Класс для управления категориями товаров.
- **Атрибуты класса**: category_count, product_count
- **Атрибуты**: name, description, __products
- **Методы**:
  - `__str__` - строковое представление категории
  - `products` (геттер) - отформатированный список товаров
  - `add_product` - добавление товара в категорию

## Новые возможности (Задание 4)

### ✅ Ограничение сложения товаров
Реализована возможность складывать только товары **одного класса**:
- `Smartphone + Smartphone` ✅
- `LawnGrass + LawnGrass` ✅
- `Smartphone + LawnGrass` ❌ (ошибка TypeError)

Для проверки используется функция `type()`.

### ✅ Ограничение добавления в категорию
Метод `add_product` принимает только объекты класса `Product` и его наследников:
- Можно добавить: `Product`, `Smartphone`, `LawnGrass`
- Нельзя добавить: строки, числа, другие объекты (ошибка TypeError)

Для проверки используется функция `isinstance()`.

### ✅ Корректный подсчет товаров
- `product_count` (атрибут класса) считает общее количество **позиций** товаров
- `__str__` категории показывает сумму `quantity` (реальное количество единиц)

## Пример использования

```python
# Создание продуктов
smartphone = Smartphone("S23", "Смартфон", 100000, 5, 95.5, "S23", 256, "Черный")
grass = LawnGrass("Газон", "Трава", 500, 20, "Россия", "7 дней", "Зеленый")

# Создание категории
category = Category("Электроника", "Описание")

# Добавление продуктов
category.add_product(smartphone)  # OK
# category.add_product("строка")  # TypeError

# Сложение продуктов
total = smartphone + smartphone  # OK
# total = smartphone + grass      # TypeError
```
